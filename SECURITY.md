# Security Policy

## Supported Versions

Only the `main` branch is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

- **Email:** mbundy15@gmail.com

Please include:
- A detailed description of the issue
- Steps to reproduce
- Any potential impact or suggestions for mitigation

We will acknowledge receipt within 72 hours and provide an estimated timeline for remediation.
